</ Sobre Mim>

Atualmente, sou desenvolvedor na Consistem, onde aprendo e cresço na área de desenvolvimento de software com a equipe de Contábil/Fiscal, utilizando a linguagem Caché. Na Consistem, adquiri conhecimentos em eSocial, SPED Fiscal, APIs, entre outros. Antes de ingressar na área, trabalhei como freelancer, desenvolvendo um sistema de gerenciamento de pedidos internos com o objetivo de organizar os pedidos para que fossem atendidos e entregues nos dias certos. Meu objetivo com o projeto era uma experiência prática tanto em organização e desenvolvimento, mas também para ter um bom projeto para currículo, sendo desenvolvido com Node.js, JavaScript e MongoDB.
